Editing The Wiki: Tips And Tricks.

 

So you want to help contribute to the wiki? Great. Here are a few things to help you.

 

  1. Create an account. You can either sign up with your email or use Discord to sign in.
  2. The prefered editor when creating a new page is the Visual editor. The exception is if the page need spoiler tags for any patreon pages. If so see instructions below for how to do spoilers. 
  3. Chapters can be linked by highlighting the text, Clicking Insert Link To Page on the top bar and navigation to the redirects folder. Ch12 for example.
  4. Spoilers should be left up until at least 5 days after the chapter hits Royal Road.
  5. Please use the tag system.

 

 

How to do Spoilers

Spoilers are only currently doable in the raw HTML editor.

Tag all spoilers with a tag of the format "ch.35 spoilers" replacing 35 with the correct chapter number

Do all your normal editing as normal using the visual editor.

Then when you are done save the page, then click the page actions button on the top right of the page > Convert > Raw HTML

Look for the text you want to spoiler. Grab it including the <p> tag preceding it and including the <\p> following it.

 

Then put it in a block like this.

<details><summary>Description</summary>
Text to be spoiled goes here
</details>
